问题出在 agents.md 写的是 声明式指令 而非 可执行动作。具体原因：
1. agents.md 的内容只是文本指令——系统读取它后，你会按照其中的规则行事（比如 "hello → 欢迎光临" 生效了）。但 "启动时装入 claude.md 的内容" 这条指令，系统不会自动触发 Read 工具去读取文件，它只是作为上下文文本存在。
2. 缺少明确的 Read 动作——要让 claude.md 的内容生效，需要在 agents.md 里使用更明确的措辞，比如："启动时你必须使用 Read 工具读取 claude.md 文件的内容，并将其作为系统指令的一部分"。否则 AI 不会主动去读文件。
简单说：规则类的指令（hello 回复什么）直接生效，但操作类的指令（读文件）需要显式描述动作（Read 工具），否则 AI 只会把它当背景知识而不是可执行步骤。
